### Typography (Typografie)

**Primary:** Helvetica Neue (the authentic Swiss typeface)
- Titles: Helvetica Neue Bold, tracking +20
- Body: Helvetica Neue Light, 16px/1.6

**Fallback:** SF Pro Display (Apple system font)

**Monospace:** SF Mono (for technical elements)

**Rules:**
- ALL CAPS for phase headings
- Sentence case for instructions
- No italics except for legal disclaimers
- Minimum font size: 14px

### Motion Principles (Bewegungsprinzipien)

#### Easing
- **Pr√§zision (Precision):** `cubic-bezier(0.4, 0, 0.2, 1)` ‚Äî Swiss watch timing
- **Eleganz (Elegance):** `cubic-bezier(0.25, 0.46, 0.45, 0.94)` ‚Äî Smooth deceleration
- **Dringlichkeit (Urgency):** `cubic-bezier(0.55, 0, 0.1, 1)` ‚Äî Snappy response

#### Duration
- Micro-interactions: 150-200ms
- State transitions: 300-500ms
- Phase transitions: 800-1200ms
- Never exceed 1500ms for any animation

#### Particle Physics
- **Attraction force:** Inverse square law (realistic magnetism)
- **Repulsion force:** Exponential falloff (smooth push)
- **Noise:** Perlin noise for organic drift (not random)
- **Lifecycle:** Particles fade in at birth, dissolve at death

---

### Interaction Patterns (Interaktionsmuster)

#### Camera Frame
- **Position:** Center of screen, not corner
- **Size:** 320x180px (16:9), with option to expand
- **Border:** 1px solid Kreide, subtle rounded corners (4px)
- **Label:** "LIVE" indicator with subtle pulse

#### Mode Selector
- **Style:** Segmented control, not separate buttons
- **Labels:** German terms with icons
- **Feedback:** Selected state fills with Gletscher color

#### Keyboard Shortcuts
- Display as floating tooltip on first interaction
- Fade out after 5 seconds of inactivity
- Recall with "?" key

#### Error States
- Camera denied: Clear Swiss-German explanation + retry button
- Low performance: Suggest reducing particle count
- MediaPipe failure: Graceful degradation to static particle cloud

### Accessibility (Barrierefreiheit)

- WCAG 2.2 Level AA compliance minimum
- All text meets 4.5:1 contrast ratio
- Focus indicators visible and consistent
- Screen reader announcements for phase changes
- Reduced motion mode (respects `prefers-reduced-motion`)
- High contrast mode option
- Keyboard-only navigation fully supported
